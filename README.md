# devicectl
A nodeJS library wrapper for Xcode's devicectl utility on macOS.

## Requirements

* Xcode 15 or greater
* NodeJS 14.17.0 or greater

### Development Requirements

* NodeJS 20.9.0 or greater for Node Test Runner support
